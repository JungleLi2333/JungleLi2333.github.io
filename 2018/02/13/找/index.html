符号表（Symbol Table）

含义

存储键值对，能够插入（insert）键值对，能够通过键直接查找到值（search）

API

规定

  • 值不可为null
  • 键不可为null
  • put()将用新值替换旧值
  • get()返回null 如果没有该键
    键和值的要求
  • 值可以是任何一般类型
  • 键必须实现comparable接口，实现compareTo方法
  • 键要实现equals来比较是否相同
  • 键要实现hashCode()来保证键的唯一性
  • 键要用不可改变的类型（比如String，Integer，java.io.File,Double）

补充：如何说x,y,z是相等的

  • 自反性：x.equals(x) ==true
  • 对称性 x.equals(y)==true
  • 传递性 x.equals(y)==y.equals(z)==true ,则x.equals(y)==true
  • 非空 x.equals(null)==false
